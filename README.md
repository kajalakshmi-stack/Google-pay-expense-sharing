Google pay expense sharing system

 A python project simulating a google pay styel expense sharing where multiple friends share expense and final settlements are calculated automatically.

 Technologies used
     -Python
     -Numpy
     -PrettyTable
     -Jupyter Notebook

Features
     -Calculates individual payments 
     -Calculates final settlements
     -Suggests payments between friends
